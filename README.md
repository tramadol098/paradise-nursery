# Paradise Nursery

Final Project for IBM Developing Front-End Apps with React

A modern plant e-commerce website built with React, Redux, and Vite.

## Features
- Browse house plants by category
- - Add plants to shopping cart
  - - Manage cart items (increase/decrease quantity)
    - - Dynamic cart counter
      - - Responsive design
       
        - Built as part of the IBM Front-End Developer Professional Certificate.
